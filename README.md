## How to Build

1. Buka terminal
2. Jalankan perintah
```
gradlew build (Windows)
./gradlew build (linux)
```

## How to Run

1. Buka terminal 
2. Jalankan perintah 
```
gradlew run (Windows)
./gradlew run (linux)
```
